# AI Agent System

## Install
```bash
pip install -r requirements.txt
```

## Setup
Create `.env` file:
```
GROQ_API_KEY=your_api_key_here
```

## Run
```bash
cd src
python main.py
```

## Usage
- `summarize <text>` - Summarize text
- `what is <topic>` - Research topic
- `calculate 2+2` - Math
- `write <prompt>` - Generate content
